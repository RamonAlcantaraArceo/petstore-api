# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-07

### Added
- Initial scaffold: FastAPI application with full Petstore API implementation
- In-memory, local PostgreSQL, and AWS RDS storage modes
- Correlation ID middleware
- API key authentication middleware
- Structured JSON logging with structlog
- Repository pattern with Protocol-based abstractions
- Docker and Docker Compose support
- Helm chart for Kubernetes deployment
- GitHub Actions CI/CD workflows
- Comprehensive test suite (unit, integration, system, e2e)
- MkDocs documentation site
